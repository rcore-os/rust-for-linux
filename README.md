# Rust for Linux 驱动模块开发

## 课题目标
学生选修该课题，可以学习到如何一步一步地开发一个Rust for Linux内核驱动；逐步积累并形成学习开发文档；进而有机会为rcore-os Rust操作系统社区以及Linux社区的发展，通过编写Rust驱动等方式贡献一份自己的力量。
